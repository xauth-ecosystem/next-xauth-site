Introduction

Welcome to the official technical documentation for the XAuth Ecosystem. This platform serves as the central hub for developers and server administrators seeking robust identity solutions for Minecraft: Bedrock Edition.

Note

This ecosystem is modular. You can deploy the XAuth core independently or scale with XAuthConnect for global authentication.

Project Overview

The suite consists of multiple integrated modules designed to work seamlessly within the PocketMine-MP environment.

  • Standardized Security: Secure password hashing and 2FA.
  • Federated Identity: Full OAuth2/OIDC implementation.
  • Multi-platform: Binding players to external social platforms.

Next Steps
